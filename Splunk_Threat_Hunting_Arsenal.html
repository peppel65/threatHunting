💡 Threat Hunting Tips & Workflow

• Sysmon EC1 (ProcCreate) + EC3 (NetConnect) correlation → find C2 beacons (Empire, BumbleBee)
• OriginalFileName mismatch → masqueraded binary (e.g. update.exe with OriginalFileName=mimikatz.exe)
• CyberChef "From Hex" → decode .lnk shortcut content to reveal embedded downloader commands
• Cuckoo sandbox DNS failure (11001) → isolate DGA domain strings from sandbox output
• Don't isolate hosts during active IR unless ransomware is encrypting — monitor first
• Ensure PS Module Logging (4103), Script Block Logging (4104), Transcription Logging via GPO
• Golden Ticket: 10yr lifetime, RC4 (not AES), no preceding 4768 for user — klist mismatch
• Silver Ticket: 4624 Kerberos on target WITHOUT preceding 4769 on any DC
• Kerberoast signature: 4769 EType 0x17 for ≥5 SPNs from one user in short window
• DCSync: 4662 Properties 1131f6aa (Get-Changes) or 1131f6ad (Get-Changes-All) from non-DC
• ADCS ESC1: cert enrollment with SAN=DA-UPN → 4768 with PreAuthType 16 (PKINIT)
• PetitPotam/PrinterBug: EID 5145 with RelativeTargetName lsarpc/efsrpc/spoolss
• BYOVD: Sysmon EID 6 for known vulnerable drivers (dbutil/gdrv/RTCore64/mhyprot)
• LSASS dump: Sysmon EID 10 to lsass.exe with GrantedAccess 0x1010/0x1410/0x1438/0x1F0FFF
• NTLM Relay: 4624 NTLM where WorkstationName doesn't match the TargetUser's normal host
• Egress busting on many ports → if blocked, good Defense-in-Depth (DMZ/VLAN/egress filtering)
• Always check Logon Type first on 4624 — Type 3 = lateral, Type 9 = PtH, Type 10 = RDP
• Standard lab-style roles: File server (SMB), ticketing (Jira), wiki (Confluence), dev workstations
